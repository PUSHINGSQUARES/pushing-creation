## Reference images

Drop reference stills, plates, and mood frames into this folder. These are the visual anchors your style and storyboard draw from.

Good references: **on-set stills** from shoots you admire, **film frames** that nail the lighting or composition you're after, **texture plates** for skin tone, fabric, vehicle livery, location mood.

The upcoming `/frames-brainstorm` command will read images in this folder via Claude vision to ground your style interview in real visual material. Until then, use them as your own reference while hand-editing `style.md`.
